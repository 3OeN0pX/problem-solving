# Problem Solving Archive

LeetCode, Baekjoon, Programmers problem-solving record repository.

## Platforms

- LeetCode
- Baekjoon
- Programmers

## Goal

- Track solved problems consistently.
- Keep code, approaches, and mistakes in one place.
- Review past solutions quickly before interviews or contests.

## Suggested Directory Structure

```text
problem-solving/
├─ README.md
├─ records.md
├─ leetcode/
├─ baekjoon/
└─ programmers/
```

You can also split by difficulty or language under each platform.

## Record Format (Recommended)

Record entries are managed in `records.md`.

- Record file: [records.md](./records.md)

## Commit Convention (Optional)

- `feat: solve [platform] [problem-name]`
- `docs: update solving notes`
- `refactor: improve previous solution`
